# 📊 Zara Sales Analysis Dashboard

## 📌 Overview
Interactive **Power BI dashboard** analyzing sales performance by category, product position, promotions, and seasonality.

## 🖼️ Preview
![Dashboard](./For%20Read%20me.png)

## 🚀 Key Insights
- Jackets generate the highest revenue (~72%)
- Aisle placement drives maximum sales
- Men’s section contributes ~90% of revenue
- Promotions and seasonal sales show balanced impact

## 🛠️ Tools Used
- Power BI  
- Excel / CSV  
- DAX
